# Vue-Admin-For-Shop
基于

[Vue-Admin-Template]: （https://gitee.com/panjiachen/vue-admin-template)

商城后台管理前端项目，数据使用MockServer模拟，满足前后端分离开发

开发路线：

1. V1.0 完成项目的全部功能（当前）
2. V2.0 封装增删改查统一api，提取工具函数
3. V3.0 封装前端页面模板根据tableDate快速生成表格
